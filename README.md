This project presents a complete Data Warehouse solution built on top of the AdventureWorks 2017 transactional database. 
The goal was to design and implement a full BI pipeline, including ETL processes, OLAP cube, and analytical reporting dashboards.

## Project Scope
The project covers the full lifecycle of a Data Warehouse implementation:
- Source system: AdventureWorks 2017 (OLTP)
- Target system: Star-schema Data Warehouse
- ETL using SQL Server Integration Services (SSIS)
- OLAP using SQL Server Analysis Services (SSAS)
- Reporting: Microsoft Power BI
